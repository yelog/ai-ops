## 🔧 修复详情

### 1. CGO 编译问题 (阻塞性)

**问题**: Release workflow 中 `CGO_ENABLED=0` 导致 SQLite 无法工作

**修复**:
- `.github/workflows/release.yml`: `CGO_ENABLED: "0"` → `"1"`
- `go.mod`: Go 版本 `1.25.0` → `1.21`
- 依赖降级兼容 Go 1.21+ (gin v1.12.0 → v1.10.0)

**验证**: ✅ Server 正常启动，SQLite 数据库正常工作

---

### 2. 离线导出异步问题

**问题**: CLI 模式下 goroutine 在进程退出后终止，导出任务无法完成

**修复**: `internal/offline/exporter.go`
```go
// 添加同步模式参数
func (e *Exporter) Export(pkg *OfflinePackage, sync ...bool) {
    if synchronous {
        runExport()  // CLI 使用同步
    } else {
        go runExport()  // API 使用异步
    }
}
```

**验证**: ✅ 成功导出 481MB 离线包 (core + network 模块)
