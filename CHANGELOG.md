# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-03-31

### Added
- Initial project setup with Go backend and React frontend
- User authentication with JWT (login/register)
- Cluster management (CRUD operations)
- Deployment templates and task management
- AI assistant with conversation support (OpenAI integration)
- Offline package export/import for air-gapped environments
- SQLite storage backend
- Ant Design based dashboard UI
- i18n support (i18next)
- Makefile build orchestration
- GitHub Actions CI/CD pipelines (ci + release)
